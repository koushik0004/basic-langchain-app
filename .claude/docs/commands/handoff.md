# Handoff

Generate a compact session handoff note. Output ONLY:

## Done
- (bullet: what was completed)

## Decisions Made
- (bullet: key choices and why, 1 line each)

## Next Steps
- (bullet: ordered by priority)

## Gotchas
- (anything that will bite next session)

Max 30 lines total. No prose, no explanation.
Save as: session-notes.md in project root.
